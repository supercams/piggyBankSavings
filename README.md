Piggy Bank Savings! is a SwiftUI-based iOS application designed to help users
manage savings goals, balances,
and basic budgeting in a simple and visually engaging way.
Inspired by the concept of a traditional piggy bank, the app provides an 
approachable interface 
for tracking personal savings while also serving as a hands-on software engineering project
focused on modern iOS development.

Features
 Create and manage savings goals
 Track balances and progress
 Simple and user-friendly SwiftUI interface
 Dynamic wallpapers and custom UI styling
 Object-oriented design with MVVM concepts
 Beginner-friendly financial management experience

Technologies Used
 Swift
 SwiftUI
 Xcode
 MVVM Architecture
 iOS Development Frameworks

This project was created as both a personal finance tool 
and a learning experience in iOS application development. 
It demonstrates skills in:
 UI/UX design
 State management in SwiftUI
 Object-oriented programming
 Data handling and application structure
 Git and GitHub version control
